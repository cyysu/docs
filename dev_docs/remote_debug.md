# 开发者远程调试用户路由器的说明文档

------
### 背景：
现在很多情况下开发者把插件发布之后，在用户路由器上会出现莫名其妙的现象，但是开发者并不能在自己的路由器上复现类似的现象。这种情况下，就需要开发者能够在用户路由器上远程执行命令。
如何在用户路由器上远程执行命令？

### 一、开发者与用户建立联系
当用户使用极路由的路由器，发现自己安装某个插件导致路由器出现故障，可以联系插件的开发者，说明插件的问题

### 二、开发者生成用户授权远程调试路由器连接
开发者进入插件列表页，选择对应的插件，进入详情页，最下面有个授权管理，点击生成授权链接, 以微信、qq或者是其他方式告于用户，让用户授权
如: 生成的链接http://open.hiwifi.com/open.php?m=myapp&a=detail&sid=xxx
  
### 三、用户授权
用户在收到开发者发送的授权链接后，点击进入到授权页面，可以选择允许授权的路由器（没有安装此插件的路由器已被过滤），同时选择授权的时间（默认24小时）
用户可以在授权历史中，看见被授权的列表，并且可以随时中断开发者的授权操作
如：http://open.hiwifi.com/open.php?m=developerremote&a=auditedlist
    
### 四、开发者远程调试
开发者在插件详情页面可看见 授权的路由器列表入口，点击进去，选择需要操作的路由器，进行远程调试
注：用户授权过期或者是没有授权则没有权限远程调试
